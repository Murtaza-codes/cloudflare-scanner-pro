## 🇬🇧 English

A powerful, highly optimized, and multi-threaded CLI tool to scan and find the lowest-latency Cloudflare IPs. Perfect for bypassing network restrictions using protocols like V2Ray, Xray, Trojan, and WireGuard.

### ✨ Features
- 🌍 **Region-Specific Scanning:** Optimized ranges for China, Russia, and Global networks.
- ⚡ **High Performance:** Uses `ThreadPoolExecutor` for asynchronous, fast scanning.
- 🎨 **Beautiful UI:** Built with `rich` for colorful tables, progress bars, and interactive menus.
- 💾 **Auto-Save:** Exports the top clean IPs directly to a text file.
- 🔄 **Customizable:** Freely specify the target port (e.g., 443, 80) and the number of IPs to scan.

### ⚙️ Installation & Usage

**1. Clone the repository:**
```bash
git clone [https://github.com/murtaza-codes/cloudflare-scanner-pro.git](https://github.com/murtaza-codes/cloudflare-scanner-pro.git)
cd cloudflare-scanner-pro
```

2. Install requirements (Windows / Linux / macOS):
pip install rich

📱 For Android (Termux):
pkg update && pkg upgrade
pkg install python git
git clone [https://github.com/murtaza-codes/cloudflare-scanner-pro.git](https://github.com/murtaza-codes/cloudflare-scanner-pro.git)
cd cloudflare-scanner-pro
pip install rich

3. Run the scanner:
python pro_scanner.py
